# Mindful-Miles

   Mindful Miles is an AI-powered travel companion that creates personalized journeys based on your mood. It recommends the ideal destination, explains why it suits you, and suggests activities to refresh your mind. Travel beyond maps — experience mindfulness, emotional wellbeing, and immersive moments.
Unlike traditional travel apps that focus only on locations and logistics, Mindful Miles considers how you feel and suggests destinations, activities, and experiences that recharge your mind and soul.

🌟 Key Features:

 - Mood-based Recommendations: Tell the app how you feel, and it instantly generates a destination tailored to your emotions.
 - Personalized Activity Plans: Each recommendation comes with curated activities — whether it’s forest walks, journaling, stargazing, or beach meditation.
 - Reasoning Behind Choices: Understand why each destination is perfect for your current mood.
 - Background Music Add-on: Optional soothing music plays seamlessly in the background to enhance your travel vibe and mindfulness experience.
 - AI-Powered Experience: Currently using a mock API for demonstration, but the system is designed to integrate AI-powered suggestions in the future.

🧭 How It Works:

 - User Input: The user selects their current mood and energy level.
 - API Call: The app fetches recommendations from a mock API, which will later be replaced by Hugging Face-powered AI for more intelligent results.
 - Generate Plan: The AI generates:
     Destination
     Reason for choosing it
     Activities to do
     Duration suggestions
 - Experience: The user can read the plan, visualize their trip, and enjoy background music for a fully immersive experience.

💡 Why Mindful Miles is Different:

 - Emotion-Driven: Most travel apps rely on location, budget, or weather — Mindful Miles is built around how you feel, not just where you go.
 - Wellbeing-Focused: Combines mental wellness with travel, making it ideal for students, professionals, or anyone seeking relaxation or inspiration.
 - Immersive Add-ons: Background music and activity suggestions make the experience multi-sensory, not just informational.
 - Simple Yet Smart: Easy-to-use interface with AI-generated insights, providing personalized, meaningful journeys in seconds.

🎵 Add-ons & Enhancements:

 - Background Music: Plays smoothly based on user’s mood to create a relaxing ambiance.
 - Expandable API: Currently powered by a mock API for demonstration, designed to seamlessly integrate AI services (like Hugging Face) in the future for smarter,     mood-based, and personalized travel recommendations.

Future Features:

 - State-wise or region-specific recommendations
 - User history and mood tracking
 - Social sharing of personalized journeys

🛠 Technologies Used:

 - Frontend: HTML, CSS, JavaScript
 - AI Integration: Mock API → Hugging Face API
 - UX Features: Background music, interactive UI
